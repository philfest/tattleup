Welcome to the JBoss Migration Tool.

This tool uses 2 open source projects from JBoss, Windup and Tattletale, to analyze your application archive. Windup specifically examines the archive for potential elements within it (proprietary configuration or code for exampe), and Tattletale takes a more general tack and looks for missing jar dependencies and classes. Thus the tools are complementary, and this site is intended as an easy and automated way to use them.

To analyze your archive simply:
  1. Choose a name for your application (use alphanumeric characters only) .
  2. Upload the application's archive (ear or war).
  3. Identify the packages to be included or excluded from analysis in a colon delimited list (e.g. com.abc:org.def).
  4. Click submit.
The tool will then analyze the archive and when finished will zip up the results from both tools for downloading. To learn more about the results and how you can run both Tattletale and Windup on your own, please visit the project's on the web.
